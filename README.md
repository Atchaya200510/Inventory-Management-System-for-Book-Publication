# 📚 Inventory Management System for A2B Book Publication

This project is a Flask-based Inventory Management System developed to manage book stock, sales, and invoice generation for a publication environment.

It is designed to work offline using a local database, making it simple and efficient for small-scale usage.

---

## 🚀 Features

- 📖 Add, update, and delete books  
- 📦 Manage inventory/stock details  
- 🧾 Generate invoices (PDF format)  
- 📊 View sales data and reports  
- 👨‍💼 Admin and Staff dashboards  
- 🔍 View available books  
- 💾 Offline database support

## ▶️ How to Run the Project

1. Clone the repository

```bash
git clone <your-github-repo-link>
```

2. Navigate to the project folder

```bash
cd Inventory-Management-System
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the Flask application

```bash
python app.py
```
